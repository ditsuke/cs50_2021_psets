The CS50x OpenCourseWare


CS50 is an introductory course in Computer Sciences by Harvard University.
Taught by Prof. David J. Malan, CS50 aims to inculcate in people the temper to think algorithmically and look for efficient ways to solve problems in Computer Science.

CS50 is a 10 week course, which starts with programming fundamentals in Scratch, working it's way up to the low level C language, followed by Python, and then HTML, CSS and Javascript (where we are right now!).
Following these, there's also Flask, something I've been looking forward to getting to after finishing with the development of this project.


We will now see "Hello, World!" programs in the languages CS50 uses (C, Python, & JavaScript!), followed by a comparison of the implementations of the same, simple yet more complicated program in C and Python.

A More Complex Program!

These generate a mario-like staircase.
We're comparing the C code to Python code side by side.
This is to show the difference betwen a lower level language and a high level language.
The Python implementation is much shorter and relatively simpler.
C 
                                
#include <stdio.h>
#include <cs50.h>
    
    // Function prototypes
    void stair(int n);
    int get_int_in_range(int lower, int upper);
    
    int main(void)
    {
        int n = get_int_in_range(1, 8);
        stair(n);
        return 0;
    }
    
    void stair(int n)
    {
        int k, l, m;
    
        // Iterate n times
        for (m = 0; m < n; m++)
        {
            // Print n-m spaces " " on each row
            for (k = 0; k < n - (m + 1); k++)
            {
                printf(" ");
            }
    
            // Print l=m sharps "#" on each row
            for (l = 0; l < m + 1; l++)
            {
                printf("#");
            }
    
            // Two spaces between left and right staircases
            printf("  ");
    
            // Print l=m sharps "#" on each row
            for (l = 0; l < m + 1; l++)
            {
                printf("#");
            }
    
            // Move cursor to next line
            printf("\n");
        }
    }
    
    int get_int_in_range(int lower, int upper)
    {
        int n = 0;
    
        // Keep asking for input until a 1 <= n <= 8
        while (n < lower || n > upper)
        {
            n = get_int("Staircase height: ");
        }
        return n;
    }
Python 
                                
from cs50 import get_int

# Get height from user, keep asking until 1 <= h >= 8
while True:
    h = get_int("Height: ")
    if h >= 1 and h <= 8:
        break

# Print staircase
for r in range(h):
    print(" " * (h - r - 1) + "#" * (r + 1) + "  " + "#" * (r + 1))
