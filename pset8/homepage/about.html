About This Project

This quaint static site is my first attempt at web development. Although not perfect, I've invested a good amount of time and put in considerable attention to detail to most of its aspects. While it not comaparable to what the seasoned web developer can produce in a fraction of the time I've invested, it was an educational, satisfying and enlightening journey for me allowing me to appreciate the power and evolution of web standards, HTML and CSS in particular.
As I got deeper into learning about CSS (though I can't do much of the awesome magic I've witnessed it's capable of), I discovered just how much it has grown over the past 2 decades. For instance, did you know that CSS is now (lemire.me) a turing complete language!?

If I am to talk about any particular challenge I had to go overcome in getting to this point in my CS50 journey, it has to be time. Being somewhat prone to procrastination, I can be slow at getting things done which is why it has taken me long to wrap the course up. It has been one of the best decisions I've taken lately, and I'd like to thank Prof. Malan and Brian Yu for all their efforts and this opportunity.

Getting back to this site, in terms of the little JavaScript I've used, there are the two prominent buttons I'd like to mention first:

  • The first controls the site's accent color (honestly, it's used in too many places to be just that), and
  • The second lets the user toggle between Light and Dark themes.

While the implementation of the random accent colors is not without its flaws (you have to cycle through a few to get one where the background and the text have enough contrast), it was interesting to implement (thank stackoverflow!).
I've also used the JavaScript-based open-source Prism syntax highlighter, which uses Javascript to tokenize and highlight the code client-side (using CSS), with some custom theming to make the code snippets on the main page look the way I want.
